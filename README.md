# Ecommerce-recommendation-system

AI-based product recommendation system using Streamlit + ML







\# 🚀 Live Features



\- 🔍 Search products

\- 📂 Filter by category

\- 🧠 AI-based recommendations

\- 📊 Clean UI with Streamlit

\- ⚡ Fast and lightweight system



\---



\# 🧠 How It Works



This system uses:



\- \*\*TF-IDF Vectorization\*\* → converts product text into numerical vectors  

\- \*\*Cosine Similarity\*\* → finds similar products  

\- \*\*Content-Based Filtering\*\* → recommends based on product description
